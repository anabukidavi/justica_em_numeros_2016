Para validar os arquivos XML gerados e enviar ao CNJ, iniciar um terminal, acessar a pasta raiz 
desse projeto e executar o comando abaixo, seguindo as instruções:


1. Substitua o parâmetro "SIGLA_REGIONAL" pelo valor correto (ex: "TRT99"), conforme atributo
   "sigla_tribunal" do arquivo de configuração.
   
2. Execute o comando referente à instância que se deseja validar (1o ou 2o graus)

# Comando para validar os XMLs unificados de ambas as instâncias (útil para enviar as remessas ao CNJ em lotes de 5000 processos)
java -jar lib\replicacaoNacional\replicacao-client-1.4.1.jar SIGLA_REGIONAL output\xmls_unificados

# Comando para validar os XMLs individuais no 1o Grau (útil para identificar processos com problemas):
java -jar lib\replicacaoNacional\replicacao-client-1.4.1.jar SIGLA_REGIONAL output\1g\xmls_individuais 

# Comando para validar os XMLs individuais no 2o Grau (útil para identificar processos com problemas):
java -jar lib\replicacaoNacional\replicacao-client-1.4.1.jar SIGLA_REGIONAL output\2g\xmls_individuais 


OBS: Se o comando indicar que a versão do replicacao-client está desatualizada, baixar a versão
     mais recente em "https://www.cnj.jus.br/owncloud/index.php/s/KwPOp6wENVBS6Vi", gravar na pasta
     "lib\replicacaoNacional" e atualizar os scripts acima!
